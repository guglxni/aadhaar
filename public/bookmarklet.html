JWT Token Beautifier

Transform raw JWT tokens into a beautiful, readable format

Drag this to your bookmarks bar:

Beautify JWT Token

How to use:

  1. Drag the button above to your bookmarks bar
  2. Navigate to a page that displays a raw JWT token (like auth/verify)
  3. Click the bookmarklet in your bookmarks bar
  4. The raw token will be transformed into a beautiful, readable format!
Note: This bookmarklet works on any page that displays a raw JWT token. It automatically detects if the page contains only a JWT token and transforms it.

Auto-Beautify Method

To automatically beautify JWT tokens when viewing them in your browser, add this script tag to the end of your HTML document:

<script src="http://localhost:3002/token-interceptor.js"></script>

Or, for a production environment with a different domain:

<script src="https://yourdomain.com/token-interceptor.js"></script>
